# Nest Template Monolith

Jumpstart your monolithic application development using NestJS.

## Requirements

- `Node 16+`
- `Git 2+`
- `Yarn 1`

## Installation

1. Get the template using `npx degit`:

```bash
npx degit your-username/nest-template-monolith my_project
```

## Installation

1. Get the template using `npx degit`:

```bash
  npx degit teo-garcia/nest-template-monolith my_project
```

2. Install the dependencies:

```bash
  yarn
```

3. Run the project:

```bash
  yarn start:dev
```

## Tools

- NestJS 10.
- TypeScript 5.
- ESLint 8 + Prettier 3.
- Husky 8 + Lint Staged 13.
- Jest 29.
- Supertest 6.

## Commands

| **Command** | **Description**                          |
| ----------- | ---------------------------------------- |
| start:dev   | Run the application in development mode. |
| start:debug | Run the application in debug mode.       |
| test        | Run unit tests using Jest.               |
| test:watch  | Run Jest in watch mode.                  |
| test:cov    | Run tests and generate coverage report.  |
| test:debug  | Run Jest in debug mode.                  |
| test:e2e    | Run end-to-end tests using Jest.         |
| lint        | Lint TypeScript files.                   |
| format      | Format TypeScript files using Prettier.  |
| build       | Build the application using Nest.        |
